PHP BACKEND INSTALLATION

To get started, you need to install on your hosting our backend.

You can install backend two different ways:

A) Manual Installation:

- Unzip archive application

- make new sql database

- find folder  "PHP" ->  "Upload" -> "Application" -> "config" -> "database.php"

- open file database.php*, use your favorite text editor and change fields "hostname", "username", "database","password"  (use the access data from your database)

- go to phpmyadmin and import test database.sql file in you database

-  find folder "PHP" -> "Upload"

- using ftp client, upload all the files from "Upload" folder to your hosting 

- go to you site and login ( admin:admin)

B) Automatic installation:

- Unzip archive application

- using ftp client, upload all the files from "Upload" folder to your hosting 

- make new sql database

- go to http://yousite.com/install or http://subdomain.yousite.com/install (If you are using a subdomain) 

- during the installation process, please fill out all the required fields or correct any errors that you show site during installation*.

*Database dump, which is supplied with the application contains test data.

 

App Setting

Configuring the application

1) Open project in Android Studio

2) Find file strings.xml

3) Change strings 

<string name="uploads_folder_api">http://club.apps76.com/uploads/</string>
<string name="endpoint_api">http://club.apps76.com</string>
<string name="mix_folder_api">http://club.apps76.com/uploadsmix/</string>

to you link url backend 

Example:

<string name="uploads_folder_api">http://myapp.com/uploads/</string>
<string name="endpoint_api">http://myapp.com</string>
<string name="mix_folder_api">http://myapp.com/uploadsmix/</string>

4) Test app

 

*before testing please fill in all sections of text information in the backend or the application does not start and will generate an error
